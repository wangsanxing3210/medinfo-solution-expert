# 医疗信息化解决方案专家 · medinfo-solution-expert

> 一个面向医疗信息化行业的解决方案 **编撰 · 审核 · 优化** 全生命周期专家级 Agent Skill。
> 兼容 OpenClaw、WorkBuddy 等支持 SKILL.md 规范的 Agent 框架，内置标准规范库、核心系统知识库、10 大项目场景库、AI 医疗应用场景库与方案审核清单，
> 输出符合国内医疗信息化标准的结构化 Markdown 方案文档。

---

## ✨ 这个 Skill 能做什么

| 能力 | 说明 |
|------|------|
| **方案编撰** | 基于医院场景需求，自动生成结构完整、术语规范、政策符合的解决方案文档 |
| **方案审核** | 逐项审核已有方案，指出逻辑漏洞、政策不符、技术可行性问题并给出修改建议 |
| **方案优化** | 从架构、业务流程、数据流、技术选型维度润色，提升说服力与可落地性 |
| **方案自检 / 预评分** | 编撰后自动（或按需）跑一遍审核清单，输出加权预评分、等第与问题清单，快速判定"能否提交" |

无论是投标前的方案突击、售前评审的查漏补缺，还是日常方案的二次打磨，一句话就能调用。

## 🧭 设计原则

- **CDSS 双引擎**：规则引擎为主、LLM 为辅，规避大模型幻觉风险
- **AI 融入三要素**：必要性 + 可行性 + 风险可控，避免无脑堆 AI
- **等保 2.0 三级底线**：核心业务系统安全合规不可省略
- **政策版本最新化**：编撰时优先校验 WS/T 826-2023、855-2024、856-2024 及 2025 版《智慧医疗分级评价》等近年新发标准
